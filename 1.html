Hypertext Markup LAnguage (HTML) is the standard markup language for creating web pages and web applications. 그렇다면 웹 페이지는 무엇일까?


웹 페이지(web page)는 월드 와이드 웹 상에 있는 개개의 문서를 가리킵니다.

.gif, jpg, .Ai, .pdf, .doc, .hwp 이와 같은 확장자 포맷이 있듯이 HTML 은 .htm, .html 확장자 포맷을 가지고 있습니다. 이 html 문서는 단순히 텍스트 파일에 불과하고 웹 브라우저가 해석을 해서 구조를 통해 화면에 렌더링 해주게 되고 사용자는 View 라고 하는 스크린을 통해 접하게 되는 것입니다. 출처: https://webclub.tistory.com/608 [Web Club:티스토리]
